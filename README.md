This platform will help residents and visitors make informed dining decisions based on both official health inspection data and community reviews

## Database Seeding

1. Download the CSV dataset file from the link: https://data.cityofnewyork.us/Health/DOHMH-New-York-City-Restaurant-Inspection-Results/43nn-pn8j
2. Create an `assist` folder in the project root (if it doesn't exist)
2. rename to (`DOHMH_New_York_City_Restaurant_Inspection_Results_20251005.csv`) and put it into the `assist` folder
3. Navigate to the `Code` directory: `cd Code`
4. Run the seed script: `npm run seed`
5. After that run npm start
