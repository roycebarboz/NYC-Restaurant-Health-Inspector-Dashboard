# NYC-Restaurant-Health-Inspector-Dashboard
This platform will help residents and visitors make informed dining decisions based on both official health inspection data and community reviews

# Make USE OF INSPECTIFY API DOUCUMENT FOR NAMING CONVENTIONS

## Database Seeding

1. Create an `assist` folder in the project root (if it doesn't exist)
2. Add the CSV dataset file (`DOHMH_New_York_City_Restaurant_Inspection_Results_20251005.csv`) into the `assist` folder
3. Navigate to the `Code` directory: `cd Code`
4. Run the seed script: `npm run seed`
